# Backup-SteamScreenshots
A script to backup your Steam screenshot folders.

<br/>

### Usage
Either run the powershell script with...
```
PS C:\Temp\Scripts> .\Backup-SteamScreenshots.ps1 -DestinationPath C:\Your\Backup\Path
```

... or run the .exe file.
