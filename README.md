# Steam-Screenshot-Backup
A script to backup your Steam screenshot folders.

<br/>

### Usage
Either run the powershell script with...
```
PS C:\Temp\Scripts> .\Steam-Screenshot-Backup.ps1
```

... or run the .exe file.
